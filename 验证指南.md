## 常见问题排查

### 问题 1：工作流运行失败

**检查：**
```bash
# 查看日志
type src\logs\app.log
```

**可能原因：**
- LLM API Key 无效 → 更新 `.env` 中的 `LLM_API_KEY`
- Firecrawl API Key 无效 → 更新 `FIRECRAWL_API_KEY`
- 网络连接问题 → 检查网络

### 问题 2：向量存储失败

**检查：**
```bash
# 查看 Qdrant 目录
dir data\qdrant
```

**可能原因：**
- 权限不足 → 以管理员身份运行
- 磁盘空间不足 → 清理磁盘

### 问题 3：周报无数据

**检查：**
```bash
# 查询数据库
.venv\Scripts\python.exe -c "from src.database import Database; db = Database(); print(db.get_statistics())"
```

**可能原因：**
- 数据库为空 → 先运行工作流采集数据
- 日期范围不对 → 调整日期范围

### 问题 4：钉钉推送失败

**检查：**
```bash
# 查看环境变量
.venv\Scripts\python.exe -c "import os; from dotenv import load_dotenv; load_dotenv(); print(f'Webhook: {os.getenv(\"DINGTALK_WEBHOOK\")[:50]}')"
```

**可能原因：**
- Webhook URL 未配置 → 配置实际的 URL
- 关键词不匹配 → 确保消息包含"情报"关键词
- 网络问题 → 检查网络连接
