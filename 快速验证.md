# 快速验证指南

## 方式一：一键验证（推荐）

### Windows 用户

双击运行：
```
verify_system.bat
```

```bash
.venv\Scripts\python.exe verify_system.py
```

这个脚本会自动检查：

- ✅ 数据库连接
- ✅ 向量存储
- ✅ API 配置

- ✅ 工作流配置

## 常见问题

### Q: 验证脚本运行失败？

**A:** 确保：
1. 已激活虚拟环境
2. 已安装所有依赖：`pip install -r requirements.txt`
3. `.env` 文件配置正确

### Q: 工作流运行但无数据？

**A:** 检查：
1. Firecrawl API Key 是否有效
2. 网络连接是否正常
3. 数据源 URL 是否可访问

### Q: 周报生成失败？

**A:** 可能原因：
1. 数据库为空 → 先运行工作流采集数据
2. 日期范围内无数据 → 调整日期范围

### Q: 钉钉推送失败？

**A:** 检查：
1. Webhook URL 是否正确（不是占位符）
2. 消息是否包含关键词"情报"
3. 网络连接是否正常
