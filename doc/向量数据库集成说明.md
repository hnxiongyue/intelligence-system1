# 向量数据库集成说明

## 概述

系统已集成 Qdrant 向量数据库，实现基于语义相似度的智能去重和相似度搜索功能。

## 核心功能

### 1. 语义去重 ✅

基于文本语义相似度的智能去重，比传统哈希去重更准确：

- **传统去重**：只能识别完全相同的文本
- **语义去重**：可以识别内容相似但表述不同的情报

**示例：**
```
原情报：国家密码管理局发布新标准
新情报：国家密码管理局发布新规定
相似度：0.91 → 识别为重复 ✅
```

### 2. 相似度搜索 ✅

根据查询文本找到语义相关的历史情报：

```python
# 查询：加密算法优化
# 结果：SM4 算法优化指南（相似度 0.76）
```

### 3. 历史上下文 ✅

为 AI 分析提供相关历史情报作为上下文：

- 按分类过滤
- 支持限制数量
- 提供元数据（标题、分类、来源）

## 技术实现

### 嵌入模型

使用阿里云百炼 **text-embedding-v4** 模型：

- **向量维度**：1024
- **距离度量**：余弦相似度（Cosine）
- **优势**：
  - 中文支持优秀
  - 语义理解准确
  - API 调用简单

### 向量数据库

使用 **Qdrant** 本地模式：

- **存储路径**：`./data/qdrant`
- **集合名称**：`intelligence`
- **索引方式**：HNSW（高效近似最近邻搜索）

### 环境变量（.env）

```env

# 本地模式（默认）

# 云服务模式（可选）
# QDRANT_URL=https://your-cluster.qdrant.io
```

### 相似度阈值

可以在代码中调整相似度阈值：

```python
# 去重阈值（默认 0.85）
is_dup, dup_info = vector_store.check_duplicate(
    title=title,
    content=content,
    similarity_threshold=0.85  # 可调整
)

# 搜索阈值（默认 0.7）
similar = vector_store.search_similar(
    text=query,
    limit=5,
    score_threshold=0.7  # 可调整
)
```

## 工作流集成

### 清洗节点（clean_node.py）

在清洗节点中集成向量去重：

```python
# 1. 基础去重（哈希）
unique_data = processor.deduplicate(cleaned_data)

# 2. 向量去重（语义）
if VECTOR_STORE_ENABLED:
    for item in unique_data:
        is_dup, dup_info = vector_store.check_duplicate(
            title=item['title'],
            content=item['content']
        )
        if not is_dup:
            final_data.append(item)
```

### 保存节点（save_node.py）

在保存节点中同步向量：

```python
# 1. 保存到 SQLite
intelligence_id = database.save_intelligence(data)

# 2. 保存到向量数据库
if VECTOR_STORE_ENABLED:
    vector_store.add_intelligence(
        intelligence_id=intelligence_id,
        title=data['title'],
        content=data['content'],
        category=data['category'],
        source=data['source']
    )
```

## 使用示例

### 1. 初始化

```python
from src.vector_store import VectorStore

# 创建向量存储实例
vector_store = VectorStore()
```

### 2. 添加情报

```python
vector_store.add_intelligence(
    intelligence_id=1,
    title="国家密码管理局发布新标准",
    content="国家密码管理局发布了新的商用密码标准...",
    category="政策",
    source="国家密码管理局"
)
```

### 3. 检查重复

```python
is_dup, dup_info = vector_store.check_duplicate(
    title="国家密码管理局发布新规定",
    content="国家密码管理局最近发布了新的商用密码标准..."
)

if is_dup:
    print(f"重复情报: {dup_info['title']}")
    print(f"相似度: {dup_info['similarity']:.2f}")
```

### 4. 搜索相似情报

```python
similar = vector_store.search_similar(
    text="密码管理相关政策",
    limit=5,
    score_threshold=0.7
)

for item in similar:
    print(f"{item['title']} (相似度: {item['similarity']:.2f})")
```

### 5. 获取历史上下文

```python
# 获取政策类情报
context = vector_store.get_context(
    category="政策",
    limit=10
)

# 获取所有情报
context = vector_store.get_context(limit=20)
```

## 性能优化

对于大量数据，建议批量处理：

## 故障排除

### 问题 1：嵌入向量生成失败

**原因**：LLM API Key 无效或网络问题

**解决**：
1. 检查 `.env` 中的 `LLM_API_KEY`
2. 确认 API 余额充足
3. 检查网络连接

### 问题 2：Qdrant 初始化失败

**原因**：权限问题或路径不存在

**解决**：
1. 确保 `./data/qdrant` 目录可写
2. 检查磁盘空间
3. 尝试删除旧数据重新初始化

### 问题 3：搜索结果为空

**原因**：相似度阈值过高或数据未添加

**解决**：
1. 降低 `score_threshold`（如 0.5）
2. 确认数据已成功添加
3. 检查向量维度是否匹配
