## 📞 问题与支持

### 遇到问题时

1. 查看日志文件：`logs/`
2. 查看文档：`README.md`
3. 查看配置：`config/`
4. 搜索错误信息
5. 寻求技术支持

### 常见问题

**Q1: MCP 连接失败？**
- 检查 Node.js 是否安装
- 检查 mcp.json 配置
- 检查 API Key 是否正确

**Q2: 采集失败？**
- 检查网络连接
- 检查数据源是否可访问
- 查看错误日志

**Q3: LLM 调用失败？**
- 检查 API Key
- 检查余额
- 检查网络连接

**Q4: 推送失败？**
- 检查 Webhook URL
- 检查网络连接
- 查看钉钉机器人设置
